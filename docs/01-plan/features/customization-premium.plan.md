# 꾸미기 아이템 프리미엄 리뉴얼

## 현황 분석

### 현재 아이템 (23개)
- 닉네임 색상 9개: 단색 위주, rainbow만 특수효과
- 프로필 프레임 6개: 기본 그라데이션 테두리
- 칭호 8개: 단조로운 텍스트

### 문제점
1. 아이템 수가 적어 수집 욕구 부족
2. 단색 닉네임 색상은 차별화 부족 - 그라데이션/글로우/애니메이션 효과 필요
3. 프레임 종류 부족, 테마 다양성 없음
4. 칭호가 평범하고 재미없음 - 메이플 세계관 반영 부족
5. 등급별 시각적 차별화 약함
6. 프리뷰가 없어서 구매 전 매력을 못 느낌

## 기술 구현

### 1단계: DB 아이템 데이터 교체
- 기존 아이템 비활성화 (is_active = 0)
- 새 아이템 73개 INSERT

### 2단계: CSS 효과 시스템 확장
- `value` 필드 규칙:
  - `#hex` = 단색
  - `gradient:색1,색2` = 그라데이션
  - `glow:색` = 발광 효과
  - `rainbow|aurora|hologram|inferno|galaxy` = 애니메이션
- StyledName.jsx에서 value 파싱 → 적절한 CSS 클래스/스타일 적용
- 새 프레임 CSS 추가 (white, black, wood, steel, rosegold, sakura, ocean, lightning, poison, dragon, starlight, bosskiller)

### 3단계: 칭호 등급별 시각 차별화
- common: 회색 배경
- rare: 파란 배경 + 약한 광택
- epic: 보라 배경 + 광택
- legendary: 금 배경 + 빛남 애니메이션

### 4단계: 꾸미기 탭 UI 개선
- 아이템 프리뷰를 실제 효과로 보여주기
- 보유/미보유 구분 강화
- 등급 필터 추가
